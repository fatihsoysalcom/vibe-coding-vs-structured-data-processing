# Vibe Coding vs Structured Data Processing

This example demonstrates the difference between 'vibe coding' (quick, unplanned solutions) and a more structured, maintainable approach. It shows how a simple data processing task can be implemented with minimal thought, leading to hidden issues, versus a robust implementation that prioritizes error handling, clarity, and maintainability.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
